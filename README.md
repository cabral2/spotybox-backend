# Spotybox

## Setup

### Install dependencies and run project

Install the dependencies

```
npm install
```

Run the project

```
npm start
```

Run the project with Hot Reload

```
npm run start:dev
```

You can test the API by going to: [localhost:3000/healthcheck](http://localhost:3000/healthcheck)

Expeted

```JSON
{
  "title":"Spotybox API",
  "version":"0.0.1"
}
```
